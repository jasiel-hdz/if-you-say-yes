# Dear Ana

Una carta editorial en forma de sitio estático.

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Personalizar

1. Edita el texto en `lib/data/letter.json`
2. Sustituye las fotos en `public/photos/` (blanco y negro, verticales idealmente)

## Stack

Next.js · React · TypeScript · Tailwind CSS · Framer Motion
